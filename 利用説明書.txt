札幌市電 輪軸管理システム v1.0 利用方法
========================================

■ 画面構成

  左側メイン : 200形・250形グループの車両カード一覧
  右側パネル : 作業計画ログ（操作履歴が自動記録）
  上部ヘッダー: 年度切替・セーブ／ロード・リセット


■ 基本操作 ─ 輪軸を動かす（交換・移設）

  1. 対象車両の「車体分離」ボタンを押す（分離理由を選択）
  2. 輪軸チップをドラッグ＆ドロップで移動
       車両スロット → 予備庫
       予備庫       → 車両スロット
       スロット同士 → 2本が自動入れ替え
       どこからでも廃棄軸エリアへドロップ → 廃棄
  3. 輪軸が4本揃ったら「車体結合」ボタンを押す

  ※ 異なるグループ（200形↔250形）の輪軸は装着不可。
  ※ 車体分離せずにはスロット操作できません。


■ 輪軸台帳（詳細モーダル）

  輪軸チップをクリックすると台帳が開きます。

  溶接状態（輪心）: 上輪・下輪ごとに「なし」「溶接あり」を選択
  軸部課題        : スリーブ有軸・乙軸を追加／削除。自由記載も可
  移設履歴        : 過去の装着場所が年度付きで記録


■ 輪軸チップの色の見方

  緑（良）: 溶接なし・健全
  赤（溶）: 輪心溶接あり

  上段 = 上輪（奇数位）、下段 = 下輪（偶数位）
  中央軸部: 橙＝スリーブ有軸、紫＝乙軸
  黄丸「!」バッジ＝自由記載課題あり


■ 年度管理

  ヘッダーの ◀ / ▶ で年度を切り替え。
  操作ログには実行時の年度が自動付与されます。


■ セーブ・ロード

  💾 セーブ  : JSON ファイルをダウンロード（日付入りファイル名）
  📂 ロード  : 保存した JSON を読み込む
  Ctrl+Z     : 操作の取り消し（最大50手）
  Ctrl+Y     : やり直し


■ 作業計画エクスポート

  右パネル下の「TXT エクスポート」で作業ログをテキスト出力。


■ 廃車処理

  車両カードの「廃車」ボタン → 確認後、装着中の輪軸は全て予備庫に自動回収。
